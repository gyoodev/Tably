# Tably – Restaurant Booking App

Flutter + Firebase приложение за резервация на маси в ресторанти.

## Особености

- Регистрация и вход
- 3 роли: Клиент, Бизнес, Админ
- Резервации
- Чат между клиент и бизнес
- Отзиви и рейтинг
